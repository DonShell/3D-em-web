Interface 3D simples estilo jogo FPS com chão e céu feita em JavaScript HTML e CSS.

No html5 temos o atributo "transform" podemos habilitalo no modo 3d
com isso podemos girar horizontalmente uma imagem, texto ou div
usando alguns comandos css como: "translateZ([distancia]px)", "rotateY([graus]deg)"...
com isso vc pode criar um ambiente 3d onde o ponto de perspectiva é a div maior, que carrega o atributo "perspective" que muda a forma com que o 3d é visto..

Para as animaçoes de arrastar para o lado foram usados funcoes simples no JQuery que incluiam rodar a div maior de acordo com as cordenadas do mouse.
